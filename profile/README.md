# Hi there 👋

Welcome to **EShipJet**.

We build commercial shipping, logistics, carrier-integration, and enterprise automation software for businesses that need reliable shipment processing, document handling, tracking, and operational visibility.

## 🙋‍♀️ About Us

**EShipJet** provides technology solutions for modern shipping and supply-chain operations. Our products help businesses manage shipments, carrier connections, labels, documents, tracking updates, and customer-specific logistics workflows.

Our main platform includes:

- **ShipperCopilot 6.4**
- Angular-based enterprise shipping frontend
- Node.js backend services
- Carrier API integrations
- Shipment tracking and scheduler services
- Printer and label automation tools
- Customer-specific deployment and support workflows

## 🚀 What We Build

We work on commercial software covering:

- Shipment creation and processing
- Carrier rate, ship, void, and tracking integrations
- FedEx, UPS, DHL, Forward Air, ATS, and other carrier workflows
- Shipping label and document generation
- ZPL, PDF, and raw printer support
- POD and delivery document handling
- Enterprise customer configuration
- On-premise and cloud deployment support
- Operational dashboards and automation tools

## 🧩 Key Projects

### ShipperCopilot 6.4

## 👩‍💻 Contribution Guidelines

This is a commercial and proprietary software organization.

Most repositories are intended for internal development, customer-specific implementation, deployment support, and controlled partner collaboration.

Before contributing:

- Work only on the assigned branch or environment
- Keep Dev, Demo, QAS, customer, and Production lanes separate
- Follow existing project patterns and code style
- Do not commit credentials, customer data, build output, local caches, or generated artifacts
- Verify changes before pushing or opening a pull request
- Document customer-impacting changes clearly
