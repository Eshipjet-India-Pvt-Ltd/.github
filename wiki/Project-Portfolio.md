## Repository Metadata Checklist

Each project should define:

- Project owner
- Technical owner
- QA owner
- Release owner
- Production support owner
- Primary branch
- Dev branch
- Demo branch
- QAS branch or release branch
- Deployment target
- Runtime dependencies
- Documentation entry point
